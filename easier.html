EASIER CLI Tool Documentation

Introduction

Welcome to the documentation for the EASIER CLI Tool. A versatile CLI tool designed to simplify development and templating processes for various frameworks like React Native, React.js, Angular, and more. Make development and project setup easier!

Installation

npm install -g easier

Usage

Display Version

easier --version

Display Help

easier --help

Generate HTML File

easier generate

Create New Folder

easier create

Create New File in Subfolder

easier create-file

Display Directory Tree

easier tree

Generate React Native App

easier generate-react-native
